Kleine Schlange

This game was my very first game per-say (i think thats how its written) because its the first time i make something original for i dont think anyone has done a snake game like this where the plants make you smaller and kind of blend with the background (Im not sure thats the case for everyone since i am not great at seeing the color green some people might be able to distnguish it better) and this plant is based on a real life plant called mugwort that can result in fatality for snakes the food that makes them grow is an egg which i chose because I find it funny that snakes can actually eat eggs. The game is called kleine Schlange because its german for small snake, why did i choose this name well i think its because it helps me to associate words between english and german also it sounds cooler. The game making progress was rather uncertain because at the start i wanted to make it so that you started big and needed to reach zero points to win but in the end i chose to make it so that the plants or antifood as i called them in the code try to blend in with the background which is green because i didnt like the blue color so it was an unintended feature for a bit and then i chose 100 to be the win condition because i dont think anyone playing this game wants to go further than that but maybe if enough people ask me i would make a mode where you have to occupy the whole screen just maybe if im not too lazy

the commands for the game are:
arrow keys and wasd for movement i think anyone whos played any game ever know which key does what and if not the arrow keys are intuitive
